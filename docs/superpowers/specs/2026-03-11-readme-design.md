# README.md 設計書

## 概要

エンドユーザー向け README.md を作成する。
ミニマル構成（概要→ダウンロード→使い方→ショートカット→開発者向け）。

## 要件

- **対象読者**: エンドユーザー中心（末尾に開発者向けセクションあり）
- **言語**: 日本語のみ
- **構成**: ミニマル構成A
- **リンク**: GitHub Releases ページへのリンクを含める

## 構成

### セクション

1. **タイトル＋概要** — ツールの目的を1〜2文で説明
2. **ダウンロードと起動** — Releases から ZIP → 展開 → index.html を開く
3. **使い方** — 音声ファイル選択 → テキスト入力
4. **キーボードショートカット** — テーブル形式
5. **開発者向け** — 必要環境・セットアップコマンド
6. **ライセンス** — MIT

### ブラウザ対応

- Google Chrome 推奨（Chromium ベースのブラウザで動作）
- Firefox は未テスト（将来対応予定）

### 開発者向けセットアップコマンド

```bash
npm install
npx playwright install chromium  # テスト用ブラウザをインストール
npm run dev      # 開発サーバー起動
npm test         # テスト実行
npm run build    # ビルド（dist/ に出力）
```
