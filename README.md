# 書き取りツール

音声を聴きながらテキストを書き取る練習ができるブラウザツールです。
インストール不要、ダウンロードして開くだけで使えます。

## ダウンロードと起動

1. [Releases](https://github.com/kanjiz/dictation-checker/releases) から最新の ZIP をダウンロード
2. ZIP を展開
3. `index.html` をダブルクリックして開く

> Google Chrome 推奨（Chromium ベースのブラウザおよび Firefox で動作）

## 使い方

1. 「音声ファイルを選択してください」から音声ファイルを選ぶ
2. テキストエリアに書き取りを入力する
3. 書き取り結果を保存したいときは Ctrl + S を押す

## キーボードショートカット

Mac では `Ctrl` の代わりに `⌘`（Command）キーを使います。

| キー（Windows/Linux） | キー（Mac） | 動作 |
| --- | --- | --- |
| Ctrl + Enter | ⌘ + Enter | 再生 / 一時停止 |
| Ctrl + ← | ⌘ + ← | 10秒戻る |
| Ctrl + → | ⌘ + → | 10秒進む |
| Ctrl + S | ⌘ + S | テキストをファイルに保存 |

> ショートカットキーと戻る/進む秒数は設定から変更できます（後述）。

## 設定

`Esc` キーを 1500ms 以内に 3 回押すと設定モーダルが開きます。

| 設定項目 | 内容 |
| --- | --- |
| 設定項目 | 内容 |
| --- | --- |
| ショートカットキー | 各操作のキーと修飾キー（Ctrl/⌘）要否を変更できます |
| 戻る秒数 / 進む秒数 | seek の秒数を個別に変更できます（1〜60秒） |
| 保存 | 変更した設定を保存してモーダルを閉じます |
| 書き出し | 現在の設定を `settings.json` としてダウンロードします |
| 読み込み | `settings.json` を読み込んで設定を復元します |
| リセット | すべての設定を初期値に戻します |

設定モーダルは `Esc` キー 1 回で閉じます。

> **注意:** W, Q, T, N, Tab は修飾キーありで設定できません。ブラウザがこれらのショートカットを横取りするため、設定しても動作しません。

## 開発者向け

### 必要環境

- Node.js 24.14.0 以上

### セットアップ

```bash
npm install
npx playwright install chromium firefox  # テスト用ブラウザをインストール
npm run dev      # 開発サーバー起動
npm test         # テスト実行
npm run build    # ビルド（dist/ に出力）
```
